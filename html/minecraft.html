FAQ
How do I join? +
Add macks.duckdns.org as a server in Minecraft Java Edition. No whitelist — just connect and play. Make sure you're on a recent version.
Is it always online? +
The server runs on home hardware, so it's online most of the time but not guaranteed 24/7. The status indicator above reflects the live state.
What mods or plugins are running? +
It's a Fabric server. BlueMap runs for the live map. Everything else is kept light — ask in-game for specifics.
Are there any rules? +
Keep it decent. No griefing, no stealing, no being a nuisance. It's a small server — common sense applies.
Can I claim land? +
No formal claim system — just build somewhere sensible and respect what others have made. The map makes it easy to find empty space.
